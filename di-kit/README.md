# AWS IAM SBX Kit

## Usage

```bash
# Package the kit
sbx kit pack di-kit

# Validate the kit
sbx kit validate di-kit

# Run the kit
sbx run di-kiro . --kit di-kit

```

## Configuration

- **AWS Region**: eu-west-2
- **SSO URL**: https://uk-digital-identity.awsapps.com/start/#/
- **Network Access**:
  - `management.us-east-1.kiro.dev:443`
  - `runtime.us-east-1.kiro.dev:443`
